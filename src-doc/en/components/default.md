# Components Overview

::: tip
New to Bamboo? Start with the [Getting Started Guide](/en/guide/getting-started) for installation and setup instructions.
:::

## Display Components

Components for displaying content and visual elements.

| Component                             | Description                                                              |
| ------------------------------------- | ------------------------------------------------------------------------ |
| [Avatar](/en/components/avatar)       | User profile images, initials, or placeholders with status indicators.   |
| [Badge](/en/components/badge)         | Labels, counts, or status indicators with icons and dots.                |
| [Icon](/en/components/icon)           | SVG icon component backed by [Feather Icons](https://feathericons.com/). |
| [Indicator](/en/components/indicator) | Status indicator dots with customizable colors and positions.            |
| [Text](/en/components/text)           | Text component with size, color, and typography options.                 |

## Form Components

Interactive components for user input and forms.

| Component                                   | Description                                                         |
| ------------------------------------------- | ------------------------------------------------------------------- |
| [Button](/en/components/button)             | Button component with sizes, variants, and loading states.          |
| [Checkbox](/en/components/checkbox)         | Checkbox component with support for groups and indeterminate state. |
| [Dropdown](/en/components/dropdown)         | Dropdown menu component with configurable placement.                |
| [Dropzone](/en/components/dropzone)         | File upload component with drag and drop support.                   |
| [File Input](/en/components/file-input)     | File input component with validation and preview.                   |
| [Input](/en/components/input)               | Input component with types, sizes, variants, and validation states. |
| [Link](/en/components/link)                 | Link component with sizes and variants.                             |
| [Radio](/en/components/radio)               | Radio button component with group support.                          |
| [Search Input](/en/components/search-input) | Search input component with debouncing and clear functionality.     |
| [Toggle](/en/components/toggle)             | Toggle switch component for boolean values.                         |

## Feedback Components

Components for providing feedback and showing loading states.

| Component                                         | Description                                                     |
| ------------------------------------------------- | --------------------------------------------------------------- |
| [Loading Dots](/en/components/loading-dots)       | Animated loading dots indicator.                                |
| [Loading Ring](/en/components/loading-ring)       | Accessible loading ring component with optional status text.    |
| [Loading Spinner](/en/components/loading-spinner) | Accessible loading spinner component with optional status text. |
| [Modal](/en/components/modal)                     | Modal dialog component with overlay and focus management.       |
| [Tooltip](/en/components/tooltip)                 | Tooltip component for contextual information.                   |

## Data Display Components

Components for displaying structured data.

| Component                     | Description                                                   |
| ----------------------------- | ------------------------------------------------------------- |
| [Table](/en/components/table) | Data table component with sorting, pagination, and selection. |
