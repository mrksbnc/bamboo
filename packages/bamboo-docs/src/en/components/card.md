# Card

A card component is a flexible container that groups related content and actions. It provides a clean, organized way to display information with optional titles, descriptions, and custom content.

## Basic Usage

The component is called `bo-card` and can be used as follows:

<div class="flex gap-4">
  <bo-card title="Card Title" description="This is a card description" />
</div>

```vue
<bo-card title="Card Title" description="This is a card description" />
```

## With Custom Content

Cards can contain custom content using slots:

<div class="flex gap-4">
  <bo-card title="Custom Card">
    <p>This is custom content inside the card.</p>
    <button class="mt-2 px-4 py-2 bg-blue-500 text-white rounded">Action</button>
  </bo-card>
</div>

```vue
<bo-card title="Custom Card">
  <p>This is custom content inside the card.</p>
  <button class="mt-2 px-4 py-2 bg-blue-500 text-white rounded">Action</button>
</bo-card>
```

## Variants

The `variant` prop allows you to customize the appearance of the card. The default variant is `default`.

<div class="flex gap-4">
  <bo-card title="Default" variant="default" description="Default card style" />
  <bo-card title="Outlined" variant="outlined" description="Card with border" />
  <bo-card title="Elevated" variant="elevated" description="Card with shadow" />
</div>

```vue
<bo-card title="Default" variant="default" description="Default card style" />
<bo-card title="Outlined" variant="outlined" description="Card with border" />
<bo-card title="Elevated" variant="elevated" description="Card with shadow" />
```

## Sizes

The `size` prop allows you to customize the padding and spacing of the card. The default size is `default`.

<div class="flex gap-4">
  <bo-card title="Small" size="sm" description="Small card" />
  <bo-card title="Default" size="default" description="Default card" />
  <bo-card title="Large" size="lg" description="Large card" />
</div>

```vue
<bo-card title="Small" size="sm" description="Small card" />
<bo-card title="Default" size="default" description="Default card" />
<bo-card title="Large" size="lg" description="Large card" />
```

## Cursor

The `cursor` prop is used to control the cursor style of the card. This is useful when the card is clickable.

<div class="flex gap-4">
  <bo-card title="Auto Cursor" cursor="cursor-auto" description="Default cursor" />
  <bo-card title="Pointer Cursor" cursor="cursor-pointer" description="Clickable card" />
  <bo-card title="Not Allowed" cursor="cursor-not-allowed" description="Disabled card" />
</div>

```vue
<bo-card title="Auto Cursor" cursor="cursor-auto" description="Default cursor" />
<bo-card title="Pointer Cursor" cursor="cursor-pointer" description="Clickable card" />
<bo-card title="Not Allowed" cursor="cursor-not-allowed" description="Disabled card" />
```

## API Reference

### Props

| Prop          | Type                           | Default         | Description                                             |
| ------------- | ------------------------------ | --------------- | ------------------------------------------------------- |
| `id`          | `string`                       | `Autogenerated` | The id of the element.                                  |
| `dataTestId`  | `string`                       | `Autogenerated` | The data test id of the element.                        |
| `variant`     | `BoCardVariant`                | `'default'`     | The variant of the card.                                |
| `size`        | `BoCardSize`                   | `'default'`     | The size of the card.                                   |
| `title`       | `string`                       | -               | The title of the card.                                  |
| `description` | `string`                       | -               | The description of the card.                            |
| `cursor`      | `BoCursor`                     | -               | The cursor to use when hovering over the card.          |
| `padding`     | `BoCardPadding`                | -               | Custom padding configuration.                           |
| `role`        | `HTMLAttributes['role']`       | -               | The role of the element.                                |
| `ariaLabel`   | `HTMLAttributes['aria-label']` | -               | Defines a string value that labels the current element. |

### Slots

The card component allows custom content to be rendered using slots.

| Name      | Props | Description                        |
| --------- | ----- | ---------------------------------- |
| `default` | `-`   | The default slot for card content. |
