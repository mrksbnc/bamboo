# Divider

A divider component is used to separate content sections with a horizontal line. It provides visual separation between different parts of the interface.

## Basic Usage

The component is called `bo-divider` and can be used as follows:

<div class="flex flex-col gap-4">
  <p>Content above the divider</p>
  <bo-divider />
  <p>Content below the divider</p>
</div>

```vue
<p>Content above the divider</p>
<bo-divider />
<p>Content below the divider</p>
```

## Variants

The `variant` prop allows you to customize the color of the divider. The default variant is `neutral`.

<div class="flex flex-col gap-4">
  <bo-divider variant="neutral" />
  <bo-divider variant="primary" />
  <bo-divider variant="secondary" />
  <bo-divider variant="accent" />
  <bo-divider variant="success" />
  <bo-divider variant="warning" />
  <bo-divider variant="info" />
  <bo-divider variant="error" />
</div>

```vue
<bo-divider variant="neutral" />
<bo-divider variant="primary" />
<bo-divider variant="secondary" />
<bo-divider variant="accent" />
<bo-divider variant="success" />
<bo-divider variant="warning" />
<bo-divider variant="info" />
<bo-divider variant="error" />
```

## Orientation

The `orientation` prop allows you to set the divider as horizontal or vertical. The default orientation is `horizontal`.

<div class="flex gap-4 items-center">
  <div class="flex flex-col gap-4">
    <p>Content above</p>
    <bo-divider orientation="horizontal" />
    <p>Content below</p>
  </div>
  <div class="flex gap-4 items-center h-20">
    <p>Left content</p>
    <bo-divider orientation="vertical" />
    <p>Right content</p>
  </div>
</div>

```vue
<!-- Horizontal divider -->
<bo-divider orientation="horizontal" />

<!-- Vertical divider -->
<bo-divider orientation="vertical" />
```

## Text Content

You can add text content to the divider using the `text` prop, and control its placement with the `placement` prop.

<div class="flex flex-col gap-4">
  <bo-divider text="Start" placement="start" />
  <bo-divider text="Center" placement="center" />
  <bo-divider text="End" placement="end" />
</div>

```vue
<bo-divider text="Start" placement="start" />
<bo-divider text="Center" placement="center" />
<bo-divider text="End" placement="end" />
```

## API Reference

### Props

| Prop          | Type                           | Default         | Description                                             |
| ------------- | ------------------------------ | --------------- | ------------------------------------------------------- |
| `id`          | `string`                       | `Autogenerated` | The id of the element.                                  |
| `dataTestId`  | `string`                       | `Autogenerated` | The data test id of the element.                        |
| `variant`     | `BoDividerVariant`             | `'neutral'`     | The color variant of the divider.                       |
| `orientation` | `BoDividerOrientation`         | `'horizontal'`  | The orientation of the divider.                         |
| `placement`   | `BoDividerPlacement`           | `'center'`      | The placement of the divider text.                      |
| `text`        | `string`                       | -               | Optional text content for the divider.                  |
| `role`        | `HTMLAttributes['role']`       | -               | The role of the element.                                |
| `ariaLabel`   | `HTMLAttributes['aria-label']` | -               | Defines a string value that labels the current element. |
