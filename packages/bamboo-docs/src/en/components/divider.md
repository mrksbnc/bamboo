# Divider

A divider component is used to separate content sections with a horizontal line. It provides visual separation between different parts of the interface.

## Basic Usage

The component is called `bo-divider` and can be used as follows:

<div class="flex flex-col gap-4">
  <p>Content above the divider</p>
  <bo-divider />
  <p>Content below the divider</p>
</div>

```vue
<p>Content above the divider</p>
<bo-divider />
<p>Content below the divider</p>
```

## Variants

The `variant` prop allows you to customize the color of the divider. The default variant is `secondary`.

<div class="flex flex-col gap-4">
  <bo-divider variant="primary" />
  <bo-divider variant="secondary" />
  <bo-divider variant="destructive" />
  <bo-divider variant="warning" />
  <bo-divider variant="success" />
  <bo-divider variant="dark" />
</div>

```vue
<bo-divider variant="primary" />
<bo-divider variant="secondary" />
<bo-divider variant="destructive" />
<bo-divider variant="warning" />
<bo-divider variant="success" />
<bo-divider variant="dark" />
```

## Types

The `type` prop allows you to customize the style of the divider line. The default type is `default`.

<div class="flex flex-col gap-4">
  <bo-divider type="default" />
  <bo-divider type="dotted" />
  <bo-divider type="dashed" />
</div>

```vue
<bo-divider type="default" />
<bo-divider type="dotted" />
<bo-divider type="dashed" />
```

## Combined Styles

You can combine different variants and types:

<div class="flex flex-col gap-4">
  <bo-divider variant="primary" type="dashed" />
  <bo-divider variant="success" type="dotted" />
  <bo-divider variant="warning" type="default" />
</div>

```vue
<bo-divider variant="primary" type="dashed" />
<bo-divider variant="success" type="dotted" />
<bo-divider variant="warning" type="default" />
```

## API Reference

### Props

| Prop         | Type                           | Default         | Description                                             |
| ------------ | ------------------------------ | --------------- | ------------------------------------------------------- |
| `id`         | `string`                       | `Autogenerated` | The id of the element.                                  |
| `dataTestId` | `string`                       | `Autogenerated` | The data test id of the element.                        |
| `variant`    | `BoDividerVariant`             | `'secondary'`   | The variant of the divider.                             |
| `type`       | `BoDividerType`                | `'default'`     | The type of the divider.                                |
| `role`       | `HTMLAttributes['role']`       | -               | The role of the element.                                |
| `ariaLabel`  | `HTMLAttributes['aria-label']` | -               | Defines a string value that labels the current element. |
