# Components

The following components are available in bamboo:

| Component                                                | Description                                                                                                                     |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| [`bo-avatar`](../en/components/avatar)                   | Avatar component for displaying user profile pictures, initials, or default images with support for indicators, sizes, and more |
| [`bo-badge`](../en/components/badge)                     | Simple badge component for displaying status indicators, tags, and more with support for icons, variants, and sizes             |
| [`bo-button`](../en/components/button)                   | Button component for submitting forms and triggering actions with support for icons, variants, and more                         |
| [`bo-card`](../en/components/card)                       | Flexible container component that groups related content and actions with customizable variants and sizes                       |
| [`bo-divider`](../en/components/divider)                 | Horizontal line component for separating content sections with customizable colors and styles                                   |
| [`bo-icon`](../en/components/icon)                       | SVG icon component with multiple sizes and color variants                                                                       |
| [`bo-input`](../en/components/input)                     | Text input field component with support for icons, validation states, and pills                                                 |
| [`bo-loading-pulse`](../en/components/loading-pulse)     | Animated pulsing dots component for indicating loading states                                                                   |
| [`bo-loading-spinner`](../en/components/loading-spinner) | Animated spinner component for indicating loading states                                                                        |
| [`bo-text`](../en/components/text)                       | Typography component for rendering text with various styles and formatting options                                              |
| [`bo-textarea`](../en/components/textarea)               | Multi-line text input component with support for icons, validation, and auto-resize                                             |
