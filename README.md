Main monorepo for the Bamboo project.

## Packages

| Package                                           | Description                                                  |
| ------------------------------------------------- | ------------------------------------------------------------ |
| [`@mrksbnc/bamboo-themes`](/packages/bamboo-core) | Shared constants and shared variables for the bamboo project |
