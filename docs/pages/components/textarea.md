# Textarea

The Textarea component is a versatile form control that allows users to enter and edit multi-line text. It supports various states, sizes, variants, and can include prefix and suffix icons. By default, it is non-resizable with a height of 3 rows. The component is designed with accessibility in mind and follows a consistent style pattern with the rest of the component library.

## Basic Example

```html
<bo-textarea
	label="Enter your text"
	description="This is a description"
/>
```

<hr class="border-gray-200 dark:border-gray-700" />
<div class="flex gap-4 items-center">
  <bo-textarea label="Enter your text" description="This is a description" />
</div>

### Props

| Prop name     | Type                | Default                     | Description                               |
| ------------- | ------------------- | --------------------------- | ----------------------------------------- |
| `modelValue`  | `string`            | `''`                        | The textarea value (v-model)              |
| `placeholder` | `string`            | `''`                        | Placeholder text                          |
| `label`       | `string`            | -                           | Label for the textarea                    |
| `variant`     | `BoTextareaVariant` | `BoTextareaVariant.default` | Visual style variant                      |
| `state`       | `BoTextareaState`   | `BoTextareaState.default`   | Textarea state (error, success, etc.)     |
| `size`        | `BoTextareaSize`    | `BoTextareaSize.default`    | Size of the textarea                      |
| `disabled`    | `boolean`           | `false`                     | Whether the textarea is disabled          |
| `required`    | `boolean`           | `false`                     | Whether the textarea is required          |
| `description` | `string`            | -                           | Help text displayed below the textarea    |
| `error`       | `string`            | -                           | Error message (shown when state is error) |
| `prefixIcon`  | `Icon`              | `Icon.none`                 | Icon to display before the textarea text  |
| `suffixIcon`  | `Icon`              | `Icon.none`                 | Icon to display after the textarea text   |
| `id`          | `string`            | Autogenerated               | Unique identifier for the textarea        |
| `name`        | `string`            | -                           | Textarea name attribute                   |
| `readonly`    | `boolean`           | `false`                     | Whether the textarea is readonly          |
| `ariaLabel`   | `string`            | -                           | Accessible label for screen readers       |
| `rows`        | `number`            | `3`                         | Number of visible rows                    |
| `resizable`   | `boolean`           | `false`                     | Whether the textarea is resizable         |
| `maxlength`   | `number`            | -                           | Maximum number of characters allowed      |
| `autofocus`   | `boolean`           | `false`                     | Whether the textarea should autofocus     |

### Events

| Event name          | Description                             |
| ------------------- | --------------------------------------- |
| `update:modelValue` | Emitted when the textarea value changes |

### Types

```ts
export enum BoTextareaVariant {
	default = 'default', // Transparent background with outline
	filled = 'filled', // Filled background with outline
	underlined = 'underlined', // Underlined textarea without outline
}

export enum BoTextareaState {
	default = 'default', // Normal state
	success = 'success', // Success/valid state
	error = 'error', // Error/invalid state
	warning = 'warning', // Warning state
}

export enum BoTextareaSize {
	small = 'small', // Small textarea size
	default = 'default', // Default textarea size
	large = 'large', // Large textarea size
}
```

## Resizable

By default, the textarea is not resizable, but you can enable resizing with the `resizable` prop:

<div class="flex flex-col gap-4">
  <bo-textarea placeholder="Non-resizable (default)" />
  <bo-textarea 
    :resizable="true" 
    placeholder="Resizable textarea" 
  />
</div>

```html
<bo-textarea placeholder="Non-resizable (default)" />
<bo-textarea
	:resizable="true"
	placeholder="Resizable textarea"
/>
```
