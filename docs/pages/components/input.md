# Input

The Input component is a versatile form control that allows users to enter and edit text. It supports various states, sizes, variants, and can include prefix and suffix icons. The component is designed with accessibility in mind and follows a consistent style pattern with the rest of the component library.

## Basic Example

```html
<bo-input
	label="Enter your text"
	description="This is a description"
/>
```

<hr class="border-gray-200 dark:border-gray-700" />
<div class="flex gap-4 items-center">
  <bo-input label="Enter your text" description="This is a description" />
</div>

### Props

| Prop name        | Type             | Default                  | Description                                   |
| ---------------- | ---------------- | ------------------------ | --------------------------------------------- |
| `modelValue`     | `string`         | `''`                     | The input value (v-model)                     |
| `placeholder`    | `string`         | `''`                     | Placeholder text                              |
| `label`          | `string`         | -                        | Label for the input                           |
| `variant`        | `BoInputVariant` | `BoInputVariant.default` | Visual style variant                          |
| `state`          | `BoInputState`   | `BoInputState.default`   | Input state (error, success, etc.)            |
| `size`           | `BoInputSize`    | `BoInputSize.default`    | Size of the input                             |
| `disabled`       | `boolean`        | `false`                  | Whether the input is disabled                 |
| `required`       | `boolean`        | `false`                  | Whether the input is required                 |
| `description`    | `string`         | -                        | Help text displayed below the input           |
| `errorMessage`   | `string`         | -                        | Error message (shown when state is error)     |
| `successMessage` | `string`         | -                        | Success message (shown when state is success) |
| `warningMessage` | `string`         | -                        | Warning message (shown when state is warning) |
| `prefixIcon`     | `Icon`           | `Icon.none`              | Icon to display before the input text         |
| `suffixIcon`     | `Icon`           | `Icon.none`              | Icon to display after the input text          |
| `type`           | `string`         | `'text'`                 | HTML input type attribute                     |
| `id`             | `string`         | Autogenerated            | Unique identifier for the input               |
| `name`           | `string`         | -                        | Input name attribute                          |
| `readonly`       | `boolean`        | `false`                  | Whether the input is readonly                 |
| `ariaLabel`      | `string`         | -                        | Accessible label for screen readers           |
| `autofocus`      | `boolean`        | `false`                  | Whether the input should autofocus            |

### Events

| Event name          | Description                          |
| ------------------- | ------------------------------------ |
| `update:modelValue` | Emitted when the input value changes |

### Types

```ts
export enum BoInputVariant {
	default = 'default', // Transparent background with outline
	filled = 'filled', // Filled background with outline
	underlined = 'underlined', // Underlined input without outline
}

export enum BoInputState {
	default = 'default', // Normal state
	success = 'success', // Success/valid state
	error = 'error', // Error/invalid state
	warning = 'warning', // Warning state
}

export enum BoInputSize {
	small = 'small', // Small input size
	default = 'default', // Default input size
	large = 'large', // Large input size
}
```
